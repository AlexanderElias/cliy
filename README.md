
# Cliy
Command Line Interface Library.

## API
- `operations: Array`
	- `operation: Object`
		- `key: String`
		- `name: String`
		- `method: AsyncFunction`
- `add: AsyncFunction`
- `find: AsyncFunction`
- `execute: AsyncFunction`
- `parse: AsyncFunction`
- `has: AsyncFunction`
- `run: AsyncFunction`
